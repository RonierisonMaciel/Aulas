# Passo a passo para executar a aplicação

## Linux ou macOS

1. Rodando a aplicação no Linux ou macOS:

```bash
python -m venv .venv
```

2. Ativando o ambiente virtual:

```bash
source .venv/bin/activate
```

3. Rodando a aplicação:

```bash
sudo python main_script.py
```
